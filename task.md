This project contain some fragments that should be used to create a single thesis (see about.txt).

"articles" directory contains all papers by the thesis author, O. O. Musiienko - they can be fully reused in the thesis

"template_example" is the LaTeX template in which thesis should to be - feel free also to use the narrative style from it

"materials" contains the early unfinished attempts to compose the thesis

"experiment" contains a lot of real data prepared by the thesis author and his peers

"design" contains unfinished project on the comprehensive analyses for all Design-related OPPs accreditated in Ukraine. A big role to AI in design teaching.

The overall thesis structure should follow the Ukrainian guidelines https://zakon.rada.gov.ua/laws/show/z0155-17#Text 

The main idea - shift from classical design to AI-related design needs to change the professional training.

Consider to actively use all needful MCPs and install additional suitable, web search, academic search etc.

Create a separate firectory for the final thesis and work in it by copying and modifyind all related materials.
