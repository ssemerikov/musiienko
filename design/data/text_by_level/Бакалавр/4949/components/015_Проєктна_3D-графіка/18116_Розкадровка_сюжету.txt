Предметом вивчення дисципліни є основні підходи та методики щодо
створення розкадровки для сценаріїв різних відео жанрів.
Міждисциплінарні зв’язки:
Забезпечуючі дисципліни: Рисунок, Композиція, Шрифти і типографіка,

Забезпечувані дисципліни: Візуальні комунікації, Комп’ютерна графіка,

Змістовий модуль 1. Розкадровка для сценарію.
Змістовий модуль 2. Особливості створення розкадровки для контенту.
Заплановані Мета навчального курсу: оволодіти загальними прийомами щодо
результати створення сценарних розкадровок для різних членів зйомочної команди
навчання та за різними відео-жанрами.
Завдання навчальної дисципліни:
- надати студентам загальні знання про візуальну сценаристику та
загальні тенденції створення сучасного відео-контенту;
- надати студентам загальні прийоми створення розкадровки для
оператора, для режисера, для акторського складу;
- надати студентам загальні прийоми створення розкадровки для
короткого метру, для повнометражного кіно;
- навчити студентів принципам роботи з ідеєю та її реалізацією в відео-
контенті.

ПРОГРАМА НАВЧАЛЬНОЇ ДИСЦИПЛІНИ
Змістовий модуль 1. Розкадровка для сценарію
Тема 1. Загальні прийоми створення розкадровки
Розкадровка. Для чого потрібна розкадровка. Рекомендовані прийоми створення
розкадровки, зрозумілої для всіх членів творчого процесу. Основні аспекти, які вказуються в
розкадровці:план, локація, подія. Позначки та корекційні знаки, зрозумілі сценаристу,
режисеру, оператору. Класичний прийом створення розкадровки та спеціальні комп’ютерні
програми для цього. Особливості позначення спец ефектів в кадрі. Позначки для оператора.
Тема 2. Розкадровка і сценарій
Розкадровка – це референти для сценарію. Основні аспекти, які вказуються в
розкадровці сценарію: час, рух, план, локація, подія, звук, акценти, спец ефекти тощо.
Сценарій сценариста, сценарій оператора, сценарій режисера – загальні вимоги та
особливості передачі інформації. Розкадровка – візуальна передача ідеї сценарію для
оператора. Розкадровка та режисерська інтерпретація відеоряду. Особливості внесення
спецефектів і монтаж.
Змістовий модуль 2. Особливості створення розкадровки для контенту
Тема 3. Розкадровка для рекламного ролика з речами
Прийоми передачі візуальної інформації для короткого метру (рекламного ролика).
Особливості роботи оператора при зйомці рекламного ролика без живих істот. Розкадровка
для такого виду реклами.
Самостійна робота: За запропонованим існуючим рекламним роликом без живих
істот нарисувати розкадровку сюжету з позначками для оператора.
Практичне завдання: Створити розкадровку сюжету рекламного ролика без живих
істот (речі або типографіка)за обраною/пропонованою тематикою.
Тема 4. Розкадровка для рекламного ролика з персонажем/персонажами
Прийоми передачі візуальної інформації для короткого метру з
персонажем/персонажами. Особливості позначень в розкадровці для акторів і іншого
технічного персоналу (грим, одяг). Особливості роботи оператора при зйомці рекламного
ролика з акторським складом . Розкадровка для такого виду реклами.
Самостійна робота: За запропонованим існуючим рекламним роликом з
персонажем/персонажами нарисувати розкадровку сюжету з позначками для оператора та
акторського складу.
Практичне завдання: Створити розкадровку сюжету рекламного ролика з
персонажем/персонажами за обраною/пропонованою тематикою.
Тема 5. Розкадровка для соціального ролика
Особливості передачі візуальної інформації соціального спрямування. Звуки і музика.
Емоційний стан, який повинен викликати соціальний ролик. «Динаміка», «небезпека»,
«страх» чи «спокій», «течія життя», «відчуття радощів», «добробут» - соціальні
характеристики візуальної інформації.
Самостійна робота: За запропонованим існуючим соціальним роликом нарисувати
розкадровку сюжету з позначками для оператора.
Практичне завдання: Створити розкадровку сюжету соціального ролика за
обраною/пропонованою тематикою.
Тема 6. Розкадровка для музичного кліпу
Особливості передачі візуальної інформації у музичному кліпі. Акцент в музиці та
акценти в кадрі/перекадровці. Міміка і жести – основа візуалізації в кліпі. Відповідність сенсу
слів до сенсу подій в кадрі. Кліп-історія. Кліп-абстракція. Кліп-типографіка. Кліп-презентація.
Самостійна робота: За запропонованим існуючим музичним кліпом нарисувати
розкадровку сюжету з позначками для оператора.
Практичне завдання: Створити розкадровку сюжету музичного кліпу за
обраною/пропонованою піснею.

Тема 7. Розкадровка для фільму
Основні прийоми створення сценарію та розкадровки для повнометражного
фільму/серіалу. Пряма сюжетна лінія. Другорядні сюжетні лінії. Особливості переплетення
сюжетних ліній, що вибудовуються у загальну картину. Розкадровка епізодів і сцен. Основні
позначки та передача інформації для всіх учасників знімального майданчика. Розкадровка
для режисера та розкадровка режисера. Особливості створення розкадровки сценарію для
повного метру.
Самостійна робота: За запропонованим існуючим епізодом фільму нарисувати
розкадровку сюжету з позначками для оператора та акторського складу.
Практичне завдання: Створити розкадровку сюжету епізоду фільму (серії фільму, або
короткої історії) за обраною/пропонованою темою.
