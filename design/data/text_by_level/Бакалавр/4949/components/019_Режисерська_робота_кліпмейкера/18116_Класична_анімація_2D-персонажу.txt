ПРОГРАМА НАВЧАЛЬНОЇ ДИСЦИПЛІНИ
V семестр
Модуль І. Класична анімація 2D персонажа
Тема 1. Сценарій для анімації.
Особливості сценарію при роботі з анімацією; поетапність роботи з відеорядом; атрибути
часової лінії; референси для ілюстратора та аніматора; розробка таблиці сценарію кожної сцени.
Тема 2. Характерні особливості 2D персонажа.
Розробка персонажів та їх характерних рис: худий, товстий, високий, низький, ходить, повзає,
літає, сильний, слабкий, веселий, депресивний тощо; розробка локацій, середовища та бекграунда,
що оточує персонажів.
Тема 3. Розмовна міміка та жести 2D персонажа.
Міміка та жести персонажів під час спілкування;. міміка обличчя при вимові букв «о», «а», «і»,
«е», «у» в різних ракурсах: фас, профіль, 1/3, 2/3, за необхідністю – ззаду, з двох сторін тощо; жести
та міміка, що виявляють емоційні стани: переляк, подив, радість, вдумливість, натхнення, недовіра
тощо.
Тема 4. Ракурси та рухи 2D персонажа.
Нестандартні ракурси персонажів; несподівані ракурси: вид зверху, вид знизу в перспективі,
дивиться вниз, звисає з чогось тощо; рухи персонажів: летить, падає, лежить, сидить, дивиться через
плече, виглядає звідкись тощо; розробка необхідних анімаційних рухів, жестів, поз, ракурсів
персонажів.
Тема 5. Аніматик за сюжетом сценарію.
Розробка послідовного покадрового накладання зображень на часову лінію; види аніматиків та
програми щодо їх створення; подальше доопрацювання: фонів, атрибутів, ракурсів тощо.
Тема 6. Створення анімаційного 2D дизайн-продукту.
Прийоми створення анімованого 2D дизайну-продукту (ролик, кліп, мультфільм) за
розробленим сценарієм; рекомендовані програми створення 2D-анімації – Adobe Animate та Adobe
Flash.
