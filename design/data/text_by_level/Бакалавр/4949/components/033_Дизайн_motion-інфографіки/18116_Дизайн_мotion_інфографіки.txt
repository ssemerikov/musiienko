ПРОГРАМА НАВЧАЛЬНОЇ ДИСЦИПЛІНИ
Модуль 1. Графічний дизайн інфографіки
Тема 1. Загальні принципи інфографіки
Інфографіка – метод візуального спрощення інформації. Види інформації для якої
створюється інфографіка: огляд, навчання, пояснення, статистика, звіт, порівняння,
презентація, реклама тощо. Як використовувати тексти для візуалізації. Текстові акценти,
важливі данні, цифри, підсумки. Візуальні ефекти які дозволяють спростити складну
інформацію.
Самостійна робота: 1. Обрати тематику майбутньої інфографіки. 2. Написати
(знайти) текст, який би розповідав у спрощеній формі складні для сприйняття речі
(наприклад: хімічні та біологічні данні забруднення природи/ пояснення як відбувається
реформа у певній галузі/ пояснення місць знаходження локацій та часу проведення для
певного міського заходу тощо).
Тема 2. Графічна складова інфографіки
Графіка для інфографіки. Основні етапи створення типографіки. Шаблони та
дизайнерські рішення для створення інфографіки. Стилізація візуальних елементів.
Елементи візуальної комунікації та їх використання в інфографіці. Вирішення кольорів.
Вирішення загальної візуалізації.
Самостійна робота: Ескізування окремих візуальних композицій для інфографіки.
Практичне завдання: За допомогою графічного редактора зробити окремі візуальні
композиції, які б доповнювали написаний текст.
Тема 3. Шрифт у інфографіці
Особливості роботи зі шрифтом для створення візуалізації. Загальні принципи
сприйняття шрифтів. Вирішення шрифтів для інфографіки
Самостійна робота: Ескізування шрифтової складової інфографіки.
Практичне завдання: Розподілити написаний текст на слайди, добрати шрифти для
подання інформації (зберегти певні шрифтові елементи окремими графічними файлами).
Модуть 2. Motion-інфографіка
Тема 4. Сценарій та розкадровка для motion-інфографіки
Загальні принципи створенні розкадровки. Розкадровка для motion-інфографіки.
Особливості візуалізації статистичних даних. Особливості візуалізації персонажів в
інфографіці. Використання візуальної комунікаціі в інфографіці.
Самостійна робота: Зробити розкадровку (розподілити текст на принципові слайди) з
вказівкою рухів певних елементів та поясненням щодо подій в кадрі.
Практичне завдання: Створити візуальні композиції з окремими рухомими
елементами за розкадровкою.
Тема 5. Flash та Gif-анімація для інфографіки
Програми для створення motion. Панелі інструментів та робота з шарами. Часова
лінія та розрахунок кадрів в секунду для зручного сприйняття. Створення простих рухів
геометричних форм. Створення рухів шрифтів. Створення перетворювань форм. Робоча
поверхня та закадрові елементи.
Самостійна робота: Оволодіти загальними прийомами роботи з інструментами
графічного редактора для створення motion.
Практичне завдання: Створити motion на «титульному» кадрі, згідно задуму та
розкадровці.
Тема 6. Створення Motion-інфографіки
Особливості роботи зі створення motion-інфографіки. Послідовність та перетікання
кадрів. Додаткові елементи та елементи візуальної комунікації для удосконалення візуальних
композицій. Створення основних моментів motion-інфографіки за лінією часу.

Самостійна робота: Розробити додаткові елементи та елементи візуальної
комунікації для удосконалення візуальних композицій motion-інфографіки.
Практичне завдання: Розмістити композиції інфографіки на всіх наступних кадрах та
створити motion для рухомих елементів.
Тема 7. Формати збереження motion-інфографіки
Відео-формати, які використовуються для показу на TV. Відео-формати, які
використовуються для публікації в мережі Інтернет. Формати для motion-інфографіки в різних
сферах використання.
Самостійна робота: 1. Доопрацювати motion-інфографіку в остаточний продукт,
готовий для показу. 2. Зберегти файл у форматі *.mp4 або *.gif
Практичне завдання: Презентація створеної motion-інфографіки.
